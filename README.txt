Amazon Web Tool Themes by Ben Spiecker (benjspie)

This is a library of themes for various Amazon FC web tools to be used in Mozilla Firefox.

To use the themes:
  Move chrome folder into Firefox profile folder
    -Open Firefox and go to about:support
    -Scroll down to 'Profile Folder' and click 'Open Folder'
    -This is the directory you want to add 'chrome' to

  Enable custom styles in Firefox
    -In Firefox, go to about:config
    -Toggle toolkit.legacyUserProfileCustomizations.stylesheets to 'true'

  In userContent.css, you can edit the entry for each tool to the path of your desired theme.
    -ex. '@import "./FCResearch/purple.css";' or '@import "./Sideline/pink.css";'

  Restart Firefox
